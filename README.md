# Pisces-Explorer

**v1.1**

基于 FastAPI + Plotly 的海洋多变量交互式可视化工具，支持声速、温度、盐度、流速、风场、波浪等变量的三维体渲染与水平切片分析。

---

## 更新日志

### v1.1（2026-07-06）

**新功能**
- 3D 显示层选择：可在深度层卡片中勾选任意层参与 3D 渲染，支持全选 / 全不选 / 隔层快捷操作，配置持久化
- 时间序列播放：加载多帧数据后出现时间轴面板，支持逐帧自动播放，播放间隔可调
- 页面刷新后自动恢复上次选择的变量

**改进**
- 矢量箭头（流速 / 风速 / 波向）全部向量化重写，渲染性能大幅提升；箭头长度随量值比例缩放
- 波向（mwd）改为显示实际角度（0–360°）而非模长，色标方案改为 HSV

**修复**
- 3D 点击层与 2D 显示层错位：部分层显示时 curveNumber 与深度索引不对应，导致点击 3D 层后 2D 切片跳到错误层
- 色标滑块拇指位置错误：初始化时拇指落在数据实际范围端点而非已保存的显示范围
- 合成波向箭头消失：mwd 移出矢量变量分类后箭头逻辑失效，已通过独立路径恢复
- 盐度存在负值时声速计算触发 RuntimeWarning（`S**1.5` 对负数无效）

### v1.0（初始版本）

- 首次发布，支持 3D 体渲染、水平切片、单点剖面、两点断面

## 功能概览

- **3D 体渲染**：多深度层叠加的三维立体显示，可自由选择参与渲染的深度层
- **水平切片**：任意深度层的水平分布热力图，支持流速 / 风场 / 波向矢量箭头叠加
- **垂直剖面**：点击地图查看任意位置的深度剖面曲线
- **垂直断面**：两点连线的剖面截面图
- **时间序列**：多帧数据按时间轴播放动画
- **多变量支持**：声速（计算量）、温度、盐度、流速（uo/vo）、风速（u10/v10）、有效波高（swh）、波向（mwd）

## 启动

```bash
python run.py
```

浏览器访问 `http://127.0.0.1:8000/`，默认端口 8000，可用 `--port` 修改。

启动时预加载数据（跳过上传步骤）：

```bash
python run.py --nc_dir /path/to/data --date 20251220
```

调试模式（文件修改后自动重载）：

```bash
python run.py --debug
```

## 数据格式

支持 NetCDF（`.nc`）格式，文件需包含以下变量（不全有也可以，缺失变量自动填充 NaN）：

| 变量 | 说明 | 维度 |
|------|------|------|
| `thetao` | 位温（必须） | time × depth × lat × lon |
| `so` | 盐度（必须） | time × depth × lat × lon |
| `uo` | 东向流速 | time × depth × lat × lon |
| `vo` | 北向流速 | time × depth × lat × lon |
| `u10` | 10m 东向风速 | time × lat × lon |
| `v10` | 10m 北向风速 | time × lat × lon |
| `swh` | 有效波高 | time × lat × lon |
| `mwd_u` | 平均波向 u 分量 | time × lat × lon |
| `mwd_v` | 平均波向 v 分量 | time × lat × lon |

声速由程序根据位温和盐度使用 Chen-Millero (1977) 公式自动计算，无需预先提供。

坐标维度名支持：`latitude`/`longitude`，深度维度支持 `depth`/`deptht`/`lev`/`level`/`z_l`/`z_t`。

**时间序列模式**：文件名中须包含 8 位日期（如 `prediction_20251220.nc`），程序按文件名排序确定时间顺序。

## 界面说明

```
┌──────────────┬──────────────────────────────────┐
│              │  3D 体渲染       │  水平切片地图  │
│  左侧控制栏  ├──────────────────┴────────────────┤
│              │         垂直剖面 / 断面图          │
└──────────────┴────────────────────────────────────┘
```

所有分割线均可拖动调整比例，控制栏宽度也可拖动缩放。

### 变量选择

左侧控制栏变量区分为两组：

- **3D 变量**（有深度维）：声速、温度、盐度、流速（合成）、东向流速、北向流速
- **2D 表面变量**：风速（合成）、风速u、风速v、有效波高、波向（合成）、波向u、波向v

选中 2D 变量时，3D 面板和剖面面板自动隐藏，地图区域全屏展示。选中合成矢量变量（流速、风速、波向）时，出现**矢量密度**滑块，控制箭头疏密。

### 深度层

下拉菜单或滑块选择当前显示的深度层，水平切片和剖面图同步更新。**3D 显示层**子面板可单独控制哪些深度层参与体渲染。

### 交互模式

- **单点剖面**：点击地图任意位置，右下方显示该点的垂直剖面曲线，可拖动选点实时更新
- **两点断面**：点击两个端点，显示连线的垂直断面图，两点均可拖动调整

### 时间序列播放

上传序列数据后，出现时间轴面板，可手动切换或点击「▶ 播放」自动逐帧播放，间隔时间可在输入框中设置（秒）。

### 色标配置

可调整每个变量的显示范围、配色方案或自定义渐变颜色，配置自动保存，下次启动时恢复。

## 打包为可执行文件

Windows 下可打包为无需 Python 环境的单文件 `.exe`：

```bash
pyinstaller --onefile --name PiscesExplorer --add-data "frontend;frontend" run.py
```

macOS / Linux：

```bash
pyinstaller --onefile --name PiscesExplorer --add-data "frontend:frontend" run.py
```
